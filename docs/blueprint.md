# **App Name**: Voluntario App

## Core Features:

- Calendar View: Display a calendar view from July 18th to September 7th with available time slots (9:00-12:30 and 16:00-20:00).
- Slot Selection: Allow users to select a time slot (one-hour increments).
- Volunteer Input: Enable users to input the name of the volunteer for each selected slot.
- Multiple Volunteers: Allow multiple volunteers per slot.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6) to inspire trust and reliability.
- Background color: A very light blue (#E3F2FD) to ensure readability and a clean interface.
- Accent color: A vibrant orange (#FFB74D) to highlight selectable time slots and call-to-action buttons.
- Font: 'PT Sans' (sans-serif) for clear readability across all devices and screen sizes.
- Use simple, recognizable icons for calendar navigation and user interaction.
- A clean, tabular layout for the calendar to present time slots clearly.
- Subtle transition animations when navigating between days and time slots.
